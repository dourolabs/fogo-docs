# Squads Multisig

Squads v3 is a multisig protocol that enables secure shared ownership and management of digital assets on Fogo. The protocol allows teams, DAOs, and organizations to require multiple signatures for executing transactions, providing enhanced security and collaborative control over on-chain assets.

## Overview

Squads v3 Multisig is available on Fogo at the standard program address:

```
SMPLecH534NA9acpos4G6x7uf3LWbCAwZQE9e8ZekMu
```

This deployment maintains full compatibility with the Squads v3 protocol, allowing you to use existing Squads tooling, SDKs, and documentation with Fogo.

## Key Features

- **Multi-signature Security**: Require multiple approvals before executing transactions
- **Flexible Threshold Configuration**: Set custom approval thresholds (e.g., 2-of-3, 3-of-5)
- **Asset Management**: Manage tokens, NFTs, and other on-chain assets collaboratively
- **Transaction Proposals**: Create, review, and approve transactions before execution
- **Developer-Friendly**: Full SDK support and comprehensive documentation

## Use Cases

Squads Multisig is ideal for:

- **Treasury Management**: Secure management of organizational funds
- **DAO Operations**: Collaborative decision-making for decentralized organizations
- **Team Wallets**: Shared control over project assets
- **Protocol Upgrades**: Multi-party approval for smart contract upgrades
