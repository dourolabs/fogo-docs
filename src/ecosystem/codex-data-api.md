# Codex Blockchain Data API

Codex is a blockchain data API that provides real-time, enriched data for tokens, wallets, and transactions across 80+ networks including Solana-compatible chains.

## Key Features

Codex offers a comprehensive API for blockchain data with sub-second latency:

- **Token Data**: Access data for 60+ million tokens including price charts, trading volume, liquidity, and holder information.
- **Wallet Data**: Query balances, token holdings, and transaction history for 400+ million wallets.
- **Real-time Updates**: Get on-chain data queryable in less than 3 seconds with websocket streaming and webhook notifications.
- **Automatic Indexing**: New tokens and assets are indexed immediately after creation.

## Use Cases

Codex is well-suited for applications that need reliable blockchain data:

- Trading interfaces and charting applications
- Portfolio trackers and wallet applications
- Analytics dashboards and data aggregators
- DeFi protocols requiring on-chain data
