# Birdeye

[Birdeye](https://birdeye.so/) is a multi-chain token tracker that provides real-time market data, analytics, and trading insights across Fogo, Solana and major EVM chains.

## Features

Birdeye offers a comprehensive suite of tools for traders and developers:

- **Token Tracking**: Live prices, charts, and trading data for tokens across multiple chains
- **Wallet Analytics**: Analyze wallet holdings, transaction history, and PnL
- **Smart Money Tracking**: Follow profitable traders and track their positions
- **Large Trade Alerts**: Monitor significant trades happening across markets

## Developer APIs

Birdeye provides APIs for integrating market data into your applications. Check out the [Birdeye API documentation](https://docs.birdeye.so/) to get started with:

- REST APIs for token prices and market data
- WebSocket connections for real-time updates
- Embeddable iFrames for charts and widgets
